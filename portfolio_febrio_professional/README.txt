# Portfolio Febrio Putra Pratama

Struktur file:

- index.html = struktur halaman
- style.css = desain tampilan
- script.js = animasi dan menu mobile
- assets/ = tempat foto, logo, atau gambar project

Cara pakai:
1. Extract file ZIP ini.
2. Buka index.html di browser.
3. Ganti link project pada bagian href="#".
4. Upload ke GitHub/Vercel/Netlify.
